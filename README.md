# CustomPlayer
CustomPlayer is a music player aimed at convenience and configurability, written in Qt 6. 

The repository is currently a placeholder. Functionality, architecture, and documentation will be added incrementally as development progresses.

## Project Status
This project is in an initial, non-functional state.

No stable releases.

No guarantees of backward compatibility.
